# Data

## Raw data

Recorded trajectory data are provided by an external research partner and are
not included because they may be subject to data-sharing restrictions.

## Simulated example data

`simulated/example_simulated_trajectory.csv` contains a trajectory generated by
the project simulator and is included as a public example dataset. It uses a
real-world GPS coordinate as its spatial reference, but all subsequent movement
and trajectory observations are simulated rather than recorded ship data.

Generate and save new simulated trajectory data:

```bash
uv run ship-simulator
```

Plot the included example dataset from the project root:

```bash
uv run python examples/plot_simulated_trajectory.py
```
